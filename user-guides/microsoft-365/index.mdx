# Microsoft 365 Module

The Microsoft 365 module provides seamless integration with Microsoft's productivity suite, enabling centralized management of users, licenses, and services from within MSPortal.ai.

## Key Features

- **User Management** - Provision and manage Microsoft 365 users
- **License Management** - Track and optimize license allocation
- **Email Configuration** - Manage Exchange Online settings
- **SharePoint Integration** - Connect to document libraries and sites
- **Teams Integration** - Sync with Microsoft Teams
- **Security Management** - Monitor and configure security settings

## Getting Started

1. Connect your Microsoft 365 tenant
2. Authenticate with global administrator credentials
3. Sync users and groups
4. Configure integration settings
5. Map MSPortal users to Microsoft 365 accounts

## Common Tasks

- **User Provisioning** - Create new Microsoft 365 accounts
- **License Assignment** - Allocate licenses to users
- **Password Management** - Reset and manage passwords
- **Group Management** - Create and manage security groups
- **Email Management** - Configure mailboxes and distribution lists
