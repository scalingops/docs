The Compliance module helps organizations maintain regulatory compliance, manage security frameworks, and track audit requirements across multiple standards and regulations.

## Key Features

- **Framework Management** - Support for multiple compliance frameworks
- **Check Management** - Track and complete compliance requirements
- **Audit Trails** - Comprehensive logging and documentation
- **Evidence Collection** - Store and manage compliance evidence
- **Automated Assessments** - Schedule recurring compliance checks
- **Reporting Tools** - Generate compliance reports and certificates

## Getting Started

1. Select applicable compliance frameworks
2. Review and customize compliance requirements
3. Assign compliance checks to team members
4. Set up recurring assessment schedules
5. Monitor compliance status through dashboards

## Available Guides

- [Adding CIS Benchmarks](./Adding%20CIS%20Benchmarks) - Configure CIS security benchmarks

## Supported Frameworks

- **CIS Controls** - Center for Internet Security benchmarks
- **ISO 27001** - Information security management
- **NIST** - Cybersecurity framework
- **HIPAA** - Healthcare compliance
- **GDPR** - Data privacy regulations
- **SOC 2** - Service organization controls
- **PCI DSS** - Payment card industry standards

## Common Tasks

- **Running Assessments** - Execute compliance checks
- **Uploading Evidence** - Attach supporting documentation
- **Tracking Remediation** - Manage non-compliance issues
- **Generating Reports** - Create audit-ready documentation
- **Scheduling Reviews** - Set up recurring assessments

## Compliance Workflow

1. **Assessment** - Evaluate current compliance status
2. **Gap Analysis** - Identify non-compliant areas
3. **Remediation** - Address compliance gaps
4. **Evidence Collection** - Gather supporting documentation
5. **Review** - Validate compliance status
6. **Reporting** - Generate compliance attestations

## Check Status Types

- **Compliant** - Requirement fully met
- **Non-Compliant** - Requirement not met
- **Partial** - Partially compliant
- **Not Applicable** - Requirement doesn't apply
- **Under Review** - Assessment in progress

## Best Practices

- **Regular Assessments** - Schedule monthly or quarterly reviews
- **Documentation** - Maintain detailed evidence records
- **Ownership** - Assign clear responsibility for each check
- **Timely Updates** - Keep compliance status current
- **Continuous Improvement** - Address gaps promptly

## Reporting Features

- **Executive Dashboards** - High-level compliance overview
- **Detailed Reports** - Comprehensive compliance documentation
- **Trend Analysis** - Track compliance over time
- **Export Options** - PDF, Excel, and CSV formats
- **Audit Packages** - Bundled evidence for auditors
