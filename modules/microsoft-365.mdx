## Overview

The Microsoft 365 module provides comprehensive monitoring and management of your clients' M365 environments. Track Secure Score, manage improvement actions, and ensure security compliance across all M365 tenants.

## Improvement Actions

### Action Management
The actions table displays:
- Action name and description
- Point value/impact
- Implementation status
- User impact assessment
- Implementation difficulty

### Action Categories
- **Identity & Access**: MFA, conditional access, privileged identity
- **Data Protection**: DLP, encryption, information protection
- **Device Security**: Endpoint protection, compliance policies
- **Application Security**: Cloud app security, OAuth apps
- **Infrastructure**: Network security, audit logging

## Troubleshooting

### Common Issues
- **Sync Failures**: Check tenant permissions and connectivity
- **Missing Actions**: Verify license requirements
- **Score Discrepancies**: Allow 24-48 hours for updates
- **Access Errors**: Re-authenticate tenant connection

## Permissions

M365 module permissions:
- `read_m365` - View M365 data
- `manage_m365` - Configure M365 settings
- `sync_m365` - Trigger manual synchronization

## Prerequisites

### Microsoft Requirements
- Global Admin or Security Admin role
- Appropriate M365 licenses
- Security Center access
- API permissions granted

### Network Requirements
- Outbound HTTPS (443)
- Access to Microsoft endpoints
- Stable internet connection
