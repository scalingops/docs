## Overview

The Settings module is your central hub for configuring MSPortal.ai to match your organization's needs. Access various configuration areas to manage users, companies, integrations, compliance settings, and more.

## Configuration Categories

### Organization Settings

#### Tenant Configuration
- Organization branding and logo
- Default timezone and locale
- Business hours and holidays
- Contact information

#### Companies Management
- Add and manage client companies
- Configure company-specific settings
- Assign users to companies
- Set up company roles and permissions

#### User Management
- Create and manage user accounts
- Assign roles and permissions
- Configure access controls
- Manage user profiles

### System Configuration

#### Compliance Settings
- Configure compliance checks and groups
- Set up compliance run schedules
- Define compliance frameworks
- Manage compliance requirements

#### Integration Settings
- **PSA Integrations**
  - ConnectWise Manage
  - Halo PSA
  - Ticket synchronization settings
  - Opportunity management
- **Microsoft 365**
  - Tenant connection
  - Secure Score monitoring
  - Calendar synchronization
- **Other Integrations**
  - API access configuration
  - Webhook management

#### Billing & Subscription
- View subscription details
- Manage payment methods
- Usage tracking
- Invoice history

### Module-Specific Settings

#### Survey Settings
- Survey templates
- Email configurations
- Response tracking settings
- NPS configuration

#### Planner Settings
- Task templates
- Default assignees
- Priority levels
- Custom fields

#### Ticket Settings
- Ticket templates
- Categories and types
- PSA synchronization rules
- Auto-assignment rules

#### Report Settings
- Report templates
- Scheduling options
- Export formats
- Distribution lists

### Advanced Configuration

#### Roles & Permissions
- Create custom roles
- Define granular permissions
- Set module access
- Configure data visibility

#### Rules Engine
- Automation rules
- Workflow triggers
- Conditional actions
- Alert configurations

#### Status Lists
- Custom status options
- Workflow states
- Priority levels
- Category definitions

#### Import/Export
- Data migration tools
- Bulk import utilities
- Export configurations
- Backup management
